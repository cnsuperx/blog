Windows上使用Visual Studio2022编译boost库

1.安装visual studio2022

visual studio2022的版本对应是msvc-14.3(即v143工具集)

2.下载boost源码

下载地址：boost官网：https://www.boost.org/

3.生成构建工具

使用visual stido2022的Developer Command Prompt for VS 2022工具：

构建，生成 b2.exe ：

4.执行 b2 stage 编译

  • 参数说明：
    • toolset=msvc-14.3：指定 VS2022 编译器
    • address-model=64：生成 64 位库
    • –build-type=complete：编译所有变体（Debug/Release、静态/动态）
    • stage：仅生成库文件（.lib 和 .dll），不安装头文件

5.编译结果

  1. 临时文件（编译中间产物）​
    • 路径​​：Boost源码目录\bin.v2
  2. 最终生成的库文件
    • Boost源码目录\stage\lib
  3. 头文件
    • Boost源码目录\include\boost-<版本>

6. 在 VS2022 中配置 Boost​

  1. ​​添加包含目录​
    • 右键项目 → 属性 → ​​C/C++ → 常规 → 附加包含目录​​ → 添加 Boost 根目录：D:\boost_1_87_0
  2. 添加库目录
    • 右键项目 → 属性 → ​​链接器 → 常规 → 附加库目录​​ → 添加：D:\boost_1_87_0\stage\lib
